# Intelcost

Prueba técnica

## Despliegue en local

* Clonar repositorio

* Ejecutar `npm install` para descargar paquetes 

* Ejecutar `ng serve`. 

* Abrir en navegador `http://localhost:4200/`


## Build para producción

* Ejecutar `ng build` todo quedará en la carpeta `dist/`
